## 🚀 How to Run

### Requirements
- Python 3.9+
- Anthropic API key

### Install
```bash
pip install -r requirements.txt
```

### Set API Key
```bash
export ANTHROPIC_API_KEY="your-key-here"
```

### Launch
```bash
streamlit run app.py
```

Open: `http://localhost:8501`
